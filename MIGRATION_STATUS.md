## 🎯 需要用户支持的内容

### 1. **API Key 配置** 🔑
**优先级：高**

需要以下 API Key：
- [ ] OpenClaw API Key（用于 AI 对话）
- [ ] Coze API Key（草包助手）
- [ ] 翻译 API Key（百度翻译 / 谷歌翻译）
- [ ] 天气 API Key（和风天气 / OpenWeather）
- [ ] OCR API Key（百度 OCR / 腾讯 OCR）

**提供方式**：
```
请在后端 .env 文件中配置：
OPENCLAW_API_KEY=your_key_here
COZE_API_KEY=your_key_here
BAIDU_TRANSLATE_API_KEY=your_key_here
HEFENG_WEATHER_API_KEY=your_key_here
```

### 2. **云服务器配置** ☁️
**优先级：高**

需要确认：
- [ ] 服务器地址（当前：ws://127.0.0.1:5000）
- [ ] 是否需要部署到云端？

**推荐配置**：
