# JVS Claw 风格首页 - 草根管家 AI 助手

## 概述
JVS Claw 风格的 AI 助手首页，突出"毒舌但有用"、"秒回不墨迹"的核心特色。

## 核心特性
- 毒舌等级：1-5级（3-5级为毒舌）
- 主色调：深绿色
- 性格特征：犀利、接地气、幽默、真实、毒舌但有用

## UI 结构
1. 标题区域 - 显示 "JVS Claw 风格" + 三个标签（毒舌但有用、默认接地气、真实不虚伪）
2. 输入区域 - 用户输入框 + 发送/清除按钮
3. 响应区域 - 显示草根回复 + 时间戳

## 技术实现
- SwiftUI 原生实现
- 支持毒舌颜色定义（#FFD700、#FF6B00、#A0A3FF、#E60000、#2196F3）
- 用户画像支持（昵称、毒舌等级、风格）
