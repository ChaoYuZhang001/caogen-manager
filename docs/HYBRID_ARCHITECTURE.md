## 📱 本地层（离线功能）

### 功能清单
| 功能 | 存储方式 | 说明 |
|------|---------|------|
| 语音备忘录 | CoreData | 本地音频 + 文本 |
| 记账算账 | SQLite | 消费记录、统计 |
| 习惯追踪 | CoreData | 习惯记录、打卡 |
| 生活记录 | CoreData | 生活日志、状态 |
| 收藏笔记本 | CoreData | 收藏内容、分类 |
| 简单对话 | CoreData | 预设对话模板 |
| 健康数据 | HealthKit | 健康数据同步 |
| 快捷指令 | Intents | Siri 集成 |

### 技术实现
- **存储**: CoreData + SQLite
- **数据同步**: 当联网时，自动同步到云端
- **简单 AI**: 本地小模型或预设回复（可选）

### 优势
- ✅ 离线可用
- ✅ 响应速度超快
- ✅ 不占服务器资源
- ✅ 隐私本地化

---

## ☁️ 云端层（联网功能）

### 功能清单
| 功能 | 连接方式 | 说明 |
|------|---------|------|
| AI 对话 | WebSocket | 实时对话、流式响应 |
| 文档解析 | HTTP REST | PDF、Word、Excel |
| 翻译 | HTTP REST | 多语言翻译 |
| OCR | HTTP REST | 图片文字识别 |
| 天气查询 | HTTP REST | 实时天气 |
| 云同步 | HTTP REST | 本地数据备份 |
| 推送通知 | APNs | 远程通知 |
| 数据分析 | HTTP REST | 消费分析、习惯分析 |

### 技术实现
- **连接**: WebSocket（实时）+ HTTP REST（异步）
- **AI 集成**: OpenClaw Gateway → 草包助手（Coze）
- **数据同步**: iOS ↔ MongoDB

### 优势
- ✅ 强大 AI 能力
- ✅ 复杂任务处理
- ✅ 多设备同步
- ✅ 云端数据备份

## 🔄 智能切换逻辑

```
用户请求
    │
    ├─ 检测网络状态
    │
    ├─ 离线模式
    │   ├─ 本地功能（备忘、记账、习惯、收藏）
    │   └─ 简单对话（预设回复）
    │
    └─ 在线模式
        ├─ 简单任务 → 本地处理（快速）
        └─ 复杂任务 → 云端处理（强大）
```

### 判断逻辑
```swift
// 简单任务 → 本地
- 简单问答
- 习惯打卡
- 记账记录
- 备忘录添加

// 复杂任务 → 云端
- 文档解析
- 多轮对话
- 复杂问题
- 多模态交互
```

## 📊 技术选型

### 本地存储方案

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| CoreData | Apple 原生，性能好 | 学习曲线陡峭 | ⭐⭐⭐⭐⭐ |
| SQLite | 简单易用，跨平台 | 需要手动管理 | ⭐⭐⭐⭐ |
| Realm | 快速，易用 | 第三方库 | ⭐⭐⭐⭐ |

**推荐**: CoreData（Apple 原生，与 SwiftUI 完美集成）

### 云端通信方案

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| WebSocket | 实时双向通信 | 保持连接 | ⭐⭐⭐⭐⭐ |
| HTTP REST | 简单，无状态 | 每次请求 | ⭐⭐⭐⭐ |
| GraphQL | 灵活查询 | 复杂 | ⭐⭐⭐ |

**推荐**: WebSocket（实时对话）+ HTTP REST（异步任务）
